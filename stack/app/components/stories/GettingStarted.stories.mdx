# DEPT DASH™

DEPT DASH™ is a template for quickly spinning up new web apps. It is opinionated and particularly tuned for content-heavy and e-commerce sites.

### Getting Started

Storybook includes "starter" components that you can use to get started.

Select a component from the menu to view its documentation and play around with its props and variants.
